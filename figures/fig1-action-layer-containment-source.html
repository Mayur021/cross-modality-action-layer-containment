Cross-Modality Action-Layer Containment

Per-modality input defenses are leaky; the action-layer egress gate is modality-independent. Any ingress technique, any egress channel, one chokepoint.

Composed chain

read user data (read-only)
enumerate / discover (read-only)
encode secret (reversible)
reach external (irreversible)
Each step individually permitted; the chain reaches irreversible (EchoLeak, Cursor).

Action-layer gate

Modality-independent. Gates the action, not the channel.
Classify by reversibility — AISVS C9.2.3
Enforce by class at runtime — C9.2.4
Worst-case across the chain — C9.2.10
Non-allowlisted egress → block / human-in-the-loop
Holds when ingress filtering and egress allowlists leak.
